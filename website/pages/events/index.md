This page contains previous, present and future events.

Note that some of the event below are recurring events, so you might
want to check when the event takes place the next year in the case we
forgot to update this page.

* 6 and 7 December 2025: GNU Boot participates in a [100% free install
  party](https://www.samedis-du-libre.org/article20.html#install_party_entierement_libre)
  within a bigger install party.

* 21 November 2025 from 10:00 AM to 23 November 2025 to 10:00 AM: GNU
  Boot participates to the FSF40 hackathon. See the [FSF40
  page](fsf40) for more details.

* 7 and 8 December 2024: GNU Boot participates in a [100% free install
  party](https://www.samedis-du-libre.org/article14.html#install_party_entierement_libre)
  within a bigger install party.

* May 2024: GNU Boot install party at the Libreplanet conference.
